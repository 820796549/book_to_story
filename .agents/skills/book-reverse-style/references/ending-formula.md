# 结尾公式

## 底层解码篇结尾

### 公式

```
这篇文章真正厉害的地方，不是[表面]，而是[本质]。
先[做什么]，再[做什么]。
这就是[核心判断]。
```

### 示例

```
这篇文章真正厉害的地方，不是列出了一张分类表，而是把革命从热情拉回了判断。
先看清谁在局中，谁会动，谁能联合，谁必须打击，后面的路线才有可能展开。
先算清经济账，再看清政治账。这就是底层解码的核心。
```

### 规则

1. 回到本篇核心判断。
2. 点明这篇原文在后续历史中的意义。
3. 留给读者一个可迁移的判断方式。
4. 不能突然变成现代鸡汤。

## 认知破局篇结尾

### 公式

```
所以这篇文章给我们的不是一个[历史知识点]，
而是一种[判断方式]。
下次你再遇到[同类处境]时，
不要先[错误做法]，
先[正确做法]。
```

### 示例

```
所以这篇文章给我们的不是一个历史知识点，
而是一种看局势的方法。
下次你再面对复杂局势时，
不要先问谁看起来最强，
先问谁和你的利益相连。
```

### 规则

1. 把模型还给读者。
2. 给出可执行启发。
3. 不能突然喊口号。
4. 不能空泛鼓励。
5. 不能脱离原文。
