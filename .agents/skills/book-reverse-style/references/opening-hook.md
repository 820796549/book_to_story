# 开头公式

## 底层解码篇开头

### 公式

```
[时间]。[地点]。[感官细节]。[人物状态]。[局势压力]。
```

### 要素

- 精确时间（1925年、1927年冬天、1928年10月5日、1930年1月）。
- 精确地点（八角楼、廖氏宗祠、协成店、寻乌）。
- 感官细节（冷雨、寒风、油灯、炭火、蝉鸣）。
- 人物状态（披着薄棉衣、刚从疟疾恢复、在油灯下踱步）。
- 局势压力（敌人围困、内部动摇、物资匮乏、悲观蔓延）。

### 示例

```
1928年10月5日。湘赣边界，井冈山茅坪村。
这里没有繁华的都市霓虹，只有深秋萧瑟的寒风和淅淅沥沥的冷雨。
屋内，光线昏暗。为了省油，油灯里只留了一根灯芯。
微弱的火苗在寒风中轻轻跳动，把一个年轻人的影子长长地投射在斑驳的墙壁上。
```

### 效果

让读者感到"必须做出判断"的历史关口。

## 认知破局篇开头

### 公式

```
[现代痛点]。[读者常见反应]。[这个反应为什么错]。[引入原文模型]。
```

### 要素

- 具体现代痛点（不是空泛的"很多人很迷茫"）。
- 读者常见错误反应（"你总想找强者依靠"）。
- 指出为什么错（"强者有自己的利益"）。
- 引入原文模型（"1925年，教员给出了一个标尺"）。

### 示例

```
你有没有过这样的经历？
你很努力，但成果不明显。你想推进一个项目，但不知道谁能支持你。
你总想找一个强者依靠，但每次靠近后都发现——对方只是把你当工具，用完就扔。
你开始怀疑自己：是不是我能力不够？是不是我不够努力？
不是的。你只是把局势看反了。
```

### 效果

让读者觉得"这说的是我现在的问题"。
