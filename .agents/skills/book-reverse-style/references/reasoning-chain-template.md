# 推理链模板

## 底层解码篇推理链

### 标准模板

```
历史困局
-> 主流误判（当时大多数人怎么想错）
-> 核心问题（原文要回答什么）
-> 判断标准（用什么标准做判断）
-> 逐层分析（按分类/条件/步骤展开）
-> 结论（给出判断）
-> 历史验证（后续历史证明判断）
```

### 示例

```
1925年历史困局（孙去世、五卅、省港罢工、党内左右倾）
-> 主流误判（陈独秀只盯上层、张国焘只盯工人，都觉得自己力量不足）
-> 核心问题（谁是敌人？谁是朋友？）
-> 判断标准（经济地位决定政治态度）
-> 逐层分析（五大阶级逐一算账）
-> 结论（敌人是地主买办，领导力量是无产阶级，最接近的朋友是农民）
-> 历史验证（1927年右派分化、湖南农运、农村包围城市）
```

## 认知破局篇推理链

### 标准模板

```
现代痛点
-> 读者错误认知（读者原本怎么想错）
-> 引入原文模型（毛选提供了什么判断方式）
-> 历史材料支撑（原文如何推导这个模型）
-> 现实迁移（现代同构处境）
-> 新判断方式（读者应该怎么重新看问题）
-> 可执行启发（读者下次可以怎么做）
```

### 示例

```
职场中看不清谁支持谁反对
-> 读者错误认知：凭感觉判断，或者讨好所有人
-> 引入毛选模型：阶级分析法 -> 利益锚定模型
-> 历史材料支撑：1925年教员用经济地位判断各阶级态度
-> 现实迁移：跨部门项目中，分析每方的利益位置
-> 新判断方式：态度可以伪装，利益不会
-> 可执行启发：先画利益地图，再行动
```
