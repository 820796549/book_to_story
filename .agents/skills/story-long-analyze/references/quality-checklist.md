# 质量检查清单

## 通用检查

- [ ] 是否有明确读者问题
- [ ] 是否有原文依据
- [ ] 是否有清楚推理链
- [ ] 是否有栏目边界
- [ ] 是否有读感和传播性
- [ ] 是否有可记住的判断句
- [ ] 标题是否有点击理由
- [ ] 开头是否抓人
- [ ] 段落是否拖沓
- [ ] 关键句是否有力量
- [ ] 结尾是否收得住
- [ ] 是否有竞品照搬风险

## 底层解码篇专项

- [ ] 历史处境是否建立
- [ ] 当时误判是否点破
- [ ] 推理链是否清楚
- [ ] 历史案例是否展开
- [ ] 破局点是否明确
- [ ] 是否写成现代鸡汤

## 认知破局篇专项

- [ ] 现代痛点是否击中
- [ ] 错误认知是否点破
- [ ] 毛选模型是否引入
- [ ] 历史材料是否支撑
- [ ] 现实迁移是否合理
- [ ] 爽点句是否到位
- [ ] 是否写成历史资料总结
