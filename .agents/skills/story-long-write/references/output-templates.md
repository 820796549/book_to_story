# 输出模板

## 底层解码篇模板

```
标题：{栏目标签} | {核心问题/悬念}

## 历史开场
{建立历史压力场：时间、地点、局面、关键人物}

## 核心问题
{当时面临的核心问题是什么}

## 当时误判
{大多数人（包括当事人）当时怎么想的}

## 原文推理链
{原文的论证步骤和推理过程}

## 历史案例展开
{每个历史案例展开成独立段落/章节}

## 关键模型
{从原文提炼的判断模型}

## 破局点
{教员如何突破当时的困局}

## 历史验证
{后来的历史如何验证了这个判断}

## 收束
{回到核心判断，点破误判}
```

## 认知破局篇模板

```
标题：{栏目标签} | {现代痛点/悬念}

## 现代痛点
{读者在现实中的具体困境}

## 错误认知
{读者通常怎么想错的}

## 引入毛选模型
{从毛选原文引入对应的判断模型}

## 历史材料支撑
{原文中的历史案例作为证据}

## 现实迁移
{模型如何映射到现代场景}

## 认知转向
{读者应该换什么思路}

## 爽点句强化
{点破误判、提炼模型、迁移现实}

## 收束
{读者下次如何重新判断}
```

## 标题公式

底层解码篇：
```
底层解码篇 | {历史处境中的决策推演}——关于《{原文标题}》的现场复盘
```

认知破局篇：
```
认知破局篇 | {现代痛点}？{毛选模型教你的新判断}
```
