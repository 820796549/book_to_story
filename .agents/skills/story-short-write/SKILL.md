# story-short-write：短篇拆书文章写作

辅助短篇拆书文章的创作，从构思到成稿。

---

## 流程

1. 读取当前项目定位和栏目协议
2. 确认写作目标（单篇短文）
3. 读取原文拆解和模型提炼
4. 按栏目协议写作
5. 质量检查
6. 输出到 `05-文章生产/成稿/`
