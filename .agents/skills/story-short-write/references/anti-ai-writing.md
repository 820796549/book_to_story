# 去AI味指南

## 核心原则

删除或替换以下AI写作痕迹：
- "本文将从三个方面……"
- "综上所述……"
- "这给我们重要启示……"
- "在当今社会……"
- "我们应该辩证地看待……"
- "值得注意的是……"
- "不可否认……"
- "从某种意义上说……"

## 替换方向

| AI味表达 | 替换为 |
|---------|--------|
| 本文将从三个方面 | 直接进入问题 |
| 综上所述 | 直接给出结论 |
| 这给我们重要启示 | 直接说"这意味着" |
| 在当今社会 | 直接描述具体场景 |
| 我们应该辩证地看待 | 直接给出判断 |

## 检查点

- 每段是否以具体判断开头
- 是否有空泛过渡句
- 是否有模板化开头
- 是否有模板化结尾
- 句子是否有节奏变化
- 是否有重复句式
