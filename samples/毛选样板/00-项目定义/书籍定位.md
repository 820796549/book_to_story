# 毛选样板：书籍定位

## 基本信息

- 书名：毛泽东选集
- 作者：毛泽东
- 原文类型：经典文集
- 版本来源：人民出版社 1991 年版
- 拆解范围：第一卷（1925-1937，国内革命战争时期）
- 是否有样文：是
- 是否有竞品：是

## 一句话定位

```text
把《毛泽东选集》从历史文献，
转成现代人在资源少、局势乱、关系复杂、行动受阻时，
能用于局势判断和认知破局的系列文章。
```

## 核心价值

1. 极端弱势条件下的局势判断方法。
2. 从混乱信息中识别主要矛盾和关键力量的能力。
3. 把资源劣势转成组织优势的路径。
4. 把历史处境中的决策算法迁移到现代现实。

## 目标读者

当用户处于以下处境：

- 资源少，不知道从哪里开始。
- 局势复杂，看不清谁是朋友谁是敌人。
- 努力很多，但方向总是错。
- 总想找强者依靠，却反复失望。
- 面对复杂关系，不知道如何判断和行动。

他想通过《毛选》获得：

```text
一套能用的局势判断方式，
而不是一段历史知识。
```

## 它是什么

1. 毛选决策算法拆解系统。
2. 底层解码篇：还原历史处境中的判断链。
3. 认知破局篇：把毛选模型迁移到现代痛点。
4. 可查询、可复用、可传播的文章生产系统。

## 它不是什么

1. 不是原文摘要。
2. 不是学术考据。
3. 不是金句合集。
4. 不是普通读后感。
5. 不是纯情绪爽文。
6. 不是万能成功学。

## 栏目体系

第一阶段固定两个栏目：

### 底层解码篇

```text
定位：拆历史处境、当时误判、推理链、破局点
目标读者：想理解毛选为什么厉害的人
禁止：写成现代鸡汤
```

### 认知破局篇

```text
定位：拆现代痛点、错误认知、毛选模型、现实迁移、爽点句
目标读者：现实中有困惑、需要破局的人
禁止：写成历史资料总结
```

## 第一篇试点

第一篇：中国社会各阶级的分析

选择原因：

- 这是毛选第一卷第一篇，是整套判断体系的起点。
- 竞品有完整的底层解码篇和认知破局篇样文。
- 适合验证两个栏目的边界。

## 成功标准

第一阶段成功必须满足：

1. 书籍定位清楚。
2. 栏目协议清楚。
3. 至少一篇底层解码篇通过审查。
4. 至少一篇认知破局篇通过审查。
5. 能说明两个栏目的边界。
6. 能沉淀可复用写法。

## 暂停条件

如果出现以下情况，暂停生产：

- 定位模糊，不知道给谁读。
- 栏目混成一锅粥。
- 样文反推不到位。
- 文章无法通过审查。
- 用户看完不知道怎么用。
