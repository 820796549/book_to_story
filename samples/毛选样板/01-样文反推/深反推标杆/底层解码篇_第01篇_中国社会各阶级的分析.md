## 2. 一句话定位

```text
这篇底层解码用1925年历史困局（孙去世、五卅、省港罢工、党内左右倾），
拆出当时两大误判（陈独秀只盯上层、张国焘只盯工人），
还原教员的经济地位决定政治态度的推理链，
最终交付破局判断：真正的同盟军是农民。
```

## 3. 历史处境

文章从四个维度建立历史压力场：

### 3.1 时间节点

1925年。具体到3月（孙中山病逝）、5月（五卅运动）、6月（省港大罢工）、12月（文章发表）。

### 3.2 主要矛盾

- 外部：帝国主义屠杀（五卅惨案）、国民党右派崛起（戴季陶主义）。
- 内部：陈独秀右倾（只盯上层）、张国焘左倾（只盯工人）。

### 3.3 各方力量

- 帝国主义：血腥镇压。
- 国民党右派：举起左手打倒帝国主义，举起右手打倒共产党。
- 国民党左派：孙去世后群龙无首。
- 共产党内：两派都觉得自己力量不足。
- 工农：爆发巨大力量（1700万人参与五卅，25万工人省港罢工）。

### 3.4 为什么这篇文章必须在此时出现

```text
革命到了必须回答"谁是敌人、谁是朋友"的关口。
如果继续按旧路线走，革命必然失败。
```

## 4. 当时误判

文章指出了两种致命误判。

### 4.1 陈独秀右倾机会主义

- 误判是什么：资产阶级民主革命应由资产阶级领导，所以全力做国民党上层工作。
- 为什么会这样：传统知识分子思维，认为落后的乡村依附先进的城市。
- 后果：向国民党右派妥协退让，放弃农民。

### 4.2 张国焘左倾机会主义

- 误判是什么：只要搞工人罢工就能解决一切。
- 为什么会这样：教条主义，只盯城市。
- 后果：同样忽略农民。

### 4.3 共同点

```text
都觉得自己力量不足，却不知道该去哪里寻找力量。
```

## 5. 核心问题

- 表层问题：中国社会有哪些阶级？
- 深层问题：各阶级对革命什么态度？
- 真正问题：谁是敌人？谁是朋友？过去为什么失败？

## 6. 推理链

```text
1925年历史困局
-> 过去革命为什么总是失败（不能团结真朋友、攻击真敌人）
-> 用什么标准分辨敌友（不是口号，不是文章，而是经济地位）
-> 经济地位决定政治态度（核心逻辑起点）
-> 五大阶级逐一算账
-> 推出结论：敌人是地主买办，领导力量是无产阶级，最接近的朋友是半无产阶级和小资产阶级
-> 历史验证：1927年右派分化、农村革命、农村包围城市
```

## 7. 分类模型

文章的核心模型是：经济地位 -> 政治态度。

### 分类依据

```text
经济地位（靠什么生存、有没有生产资料、生活水平如何）
-> 政治态度（对革命是支持、观望还是反对）
-> 策略安排（敌人、朋友、领导力量、提防对象）
```

### 五大阶级 + 游民

| 阶级 | 经济地位 | 政治态度 | 定性 |
|---|---|---|---|
| 地主/买办 | 国际资产阶级附庸 | 始终反革命 | 敌人 |
| 中产阶级（民族资产阶级） | 夹缝生存，两头受气 | 矛盾动摇，必分化 | 右翼是敌，左翼是友，需提防 |
| 小资产阶级 | 小生产经济，分左中右三派 | 平时各异，战时全被裹挟 | 最接近的朋友 |
| 半无产阶级 | 极细小生产，常出卖劳动力 | 极度需要变革 | 最接近的朋友（主力） |
| 无产阶级 | 彻底失去生产资料，集中 | 特别能战斗 | 领导力量 |
| 游民无产者 | 失地失业，秘密帮会 | 勇敢但有破坏性 | 引导得法可成革命力量 |

## 8. 破局点

```text
破局前的错误路线：只盯上层（陈独秀）或只盯城市工人（张国焘）。
破局后的判断路线：农民是最大同盟军。
关键转向：从看上层人物转为看底层力量，从看城市转为看农村。
为什么有效：因为农民占人口绝大多数，经济地位决定了他们必须革命。
```

## 9. 历史验证

文章用了三个历史验证：

1. 1927年四一二政变：民族资产阶级右翼果然分化跑向反革命。
2. 湖南农民运动考察：贫农爆发惊天动地的伟力。
3. 农村包围城市道路：基于农民力量的战略选择。

验证是否合理：合理，历史事实支持。

## 13. 爽点句类型

| 类型 | 功能 | 效果 | 可复用句式 | 禁用边界 |
|---|---|---|---|---|
| 点破误判 | 指出过去为什么失败 | 读者感到原来问题在这里 | "其基本原因就是因为不能团结真正的朋友，以攻击真正的敌人。" | 不能脱离原文依据 |
| 一句话定性 | 把复杂问题简化成判断 | 读者记住核心判断 | "经济地位决定政治态度。" | 不能过度简化 |
| 翻案 | 推翻流行观点 | 读者感到认知翻转 | "陈独秀只盯着国民党高层，张国焘只盯着城市工人，他们都觉得自己力量不足。" | 不能歪曲历史 |
| 穿透句 | 把历史和现实打通 | 读者感到被击中 | "谁是我们的敌人？谁是我们的朋友？这个问题是革命的首要问题。" | 不能滥用 |
| 模型句 | 给出可迁移判断方式 | 读者获得方法 | "不可不将中国社会各阶级的经济地位及其对于革命的态度，作一个大概的分析。" | 不能变成万能公式 |

## 14. 原文依据使用方式

文章使用原文的方式：

- 直接引用：开头第一句、关键判断句、21句附录。
- 转述观点：对每个阶级的分析，不是照抄原文，而是用现代语言重述。
- 提炼结构：把原文的五类阶级整理成表格。
- 借原文做历史验证：用1927年政变和湖南农运证明原文判断。
- 超出原文的部分：历史背景描写（孙去世、五卅等）、戴季陶主义介绍。这些是背景材料，不改变原文判断。

## 17. 对栏目协议的修正

1. 底层解码篇的历史开场不能只写背景，必须制造"不得不判断"的压力场。
2. 误判必须写具体，不能只写"当时有人看错了"。
3. 推理链必须有"用什么标准判断"这个环节。
4. 必须写历史验证，不能只分析不验证。
5. 可以加附录提供原文依据，但不能让附录替代正文推理。
