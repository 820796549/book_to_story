## 2. 一句话定位

```text
这篇认知破局用毛选阶级分析法和调查方法论，
击穿现代人在职场/创业/个人成长中的错误认知（本本主义、看不清利益格局），
最终交付三个可执行工具：权力地图、去伪存真脚本、个人能量损益表。
```

## 3. 现代痛点

文章命中了以下现代痛点：

### 3.1 职场困境

- 跨部门扯皮，不知道谁支持谁反对。
- 推行新政时被老员工抵制。
- 看不清谁是真正的盟友。

### 3.2 创业困境

- 资源少，不知道怎么和巨头竞争。
- 盲目跟风风口，没有自己的判断。
- 不知道用户真正要什么。

### 3.3 个人成长困境

- 拖延症，知道该做但就是不动。
- 精神内耗，内心矛盾消耗能量。
- 学了很多方法但用不出来。

## 4. 错误认知

| 错误认知 | 它为什么有迷惑性 | 后果 | 文章如何推翻 |
|---|---|---|---|
| 努力就能改变处境 | 主流成功学都在说 | 方向错了越努力越错 | 先做调查，再判断局势 |
| 强者最值得依靠 | 看起来合理 | 反复失望，被当工具 | 用阶级分析看利益位置 |
| 数据能说明一切 | 看起来很科学 | 被虚荣指标误导 | 必须到一线做调查 |
| 跟风口就能成功 | 有成功案例 | 盲目跟风，没有根基 | 找到真正的用户痛点 |
| 拖延是因为懒 | 表面看起来是 | 自我攻击，越拖越严重 | 拖延是内部保护机制 |

## 5. 毛选模型

文章提炼了三个核心模型。

### 5.1 真相挖掘漏斗

```text
没有体感，就没有决策权。
```

逻辑链：

```text
下沉（离开办公室到一线）
-> 解剖（选取典型切片全要素分析）
-> 对质（不同信息碰撞剔除噪音）
-> 重构（基于事实重构判断）
```

### 5.2 利益锚定模型

```text
屁股决定脑袋，恐惧决定边界。
```

逻辑链：

```text
定位（查户口：看组织位置和KPI）
-> 定性（判断恐惧点和贪婪点）
-> 定策（对敌人丢石头，对中间派掺沙子，对朋友挖墙脚）
```

### 5.3 统一战线公式

```text
发展进步势力 + 争取中间势力 + 孤立顽固势力 = 胜利
```

## 6. 历史材料使用方式

文章使用历史材料的方式：

- 阶级分析法映射：把五大阶级映射到现代职场角色（地主=既得利益集团，中产阶级=中层管理者，小资产阶级=核心骨干，无产阶级=基层员工，游民=外包/投机者）。
- 调查方法论：把开调查会映射到用户研究和产品调研。
- 农村包围城市：把战略映射到华为案例。
- 统一战线：把抗日民族统一战线映射到职场合纵连横。

历史材料不是主体，而是为现代模型提供来源和重量。

## 17. 对栏目协议的修正

1. 认知破局篇必须有明确的"现代痛点"入口，不能从历史进入。
2. 必须有"错误认知"环节，指出读者原本怎么想错。
3. 毛选模型必须提炼成可迁移的模型，不能只讲历史。
4. 现实迁移必须有"同构关系"说明，不能硬套。
5. 结尾必须给出可执行工具，不能只做总结。
6. 必须区分原文依据和现代发挥。
7. 现代发挥不能过度，必须和毛选模型有结构相似性。
