# 现实迁移：中国社会各阶级的分析

## 1. 来源模型

敌友识别模型、阶级分析模型、力量判断模型。

## 2. 原文问题

1925年革命困局：谁是敌人？谁是朋友？过去为什么失败？

## 3. 现代同构处境

### 3.1 职场处境

- 跨部门项目：不知道谁支持谁反对。
- 推行改革：被老员工抵制，新人支持但没话语权。
- 站队问题：看不清派系，怕站错队。
- 资源争夺：资源少，不知道谁能合作。

### 3.2 创业处境

- 资源少，不知道怎么和巨头竞争。
- 看不清用户真正要什么。
- 找不到第一批核心用户。
- 不知道该联合谁、提防谁。

### 3.3 个人成长处境

- 努力很多但方向不对。
- 总想找强者依靠，反复失望。
- 看不清自己真正的优势和劣势。
- 不知道哪些关系值得投入。

## 4. 读者痛点

### 4.1 痛点一：看不清利益格局

```text
读者处境：跨部门项目，各方态度不明。
读者感受：不知道该信任谁，怕被坑。
读者常见反应：凭感觉判断，或者谁都不得罪。
这个反应为什么错：凭感觉会看错，谁都不得罪等于没有策略。
```

### 4.2 痛点二：努力方向错误

```text
读者处境：很努力，但成果不明显。
读者感受：挫败、迷茫。
读者常见反应：更努力。
这个反应为什么错：方向错了，越努力越远。
```

### 4.3 痛点三：总想找强者依靠

```text
读者处境：资源少，想找靠山。
读者感受：反复失望，被当工具。
读者常见反应：继续找更强的靠山。
这个反应为什么错：强者不会真正帮你，只有利益相连的人才会。
```

## 5. 读者常见误判

| 误判 | 真实问题 | 新判断 |
|---|---|---|
| 看谁喊得响就支持谁 | 口号不代表立场 | 看利益位置 |
| 强者最值得依靠 | 强者有自己的利益 | 找利益相连的人 |
| 努力就能改变 | 方向比努力重要 | 先判断局势再行动 |
| 所有人都可以争取 | 利益冲突的人不能 | 区分敌友，分别对待 |
| 态度好就是朋友 | 态度可以伪装 | 看经济地位（利益） |

## 6. 新判断方式

### 6.1 画利益地图

```text
第一步：列出所有相关方。
第二步：分析每方的利益位置（变革会获益还是受损）。
第三步：判断每方的态度（支持/观望/反对）。
第四步：制定策略（团结/争取/孤立/提防）。
```

### 6.2 找利益相连的人

```text
不要问"谁最强"。
要问"谁和我的利益相连"。
```

### 6.3 先调查再判断

```text
不要凭感觉判断局势。
先做调查：找一线的人、找真实数据、找失败案例。
```

## 7. 可执行启发

1. 下次遇到复杂局势，先画利益地图，不要凭感觉。
2. 不要总想找强者，先找利益相连的人。
3. 方向比努力重要，先判断局势再行动。
4. 不是所有人都可以争取，区分对待。
5. 态度可以伪装，利益不会。

## 8. 爽点句方向

| 类型 | 方向 | 可复用句式 |
|---|---|---|
| 点破误判 | 你不是不努力，你是把局势看反了 | "你不是XX，你是XX。" |
| 翻案 | 强者不是靠山，利益相连的人才是 | "真正的XX不是XX，而是XX。" |
| 模型句 | 先画利益地图，再行动 | "先XX，再XX。" |
| 穿透句 | 态度可以伪装，利益不会 | "XX可以伪装，XX不会。" |

## 9. 迁移边界

1. 不能把现代职场关系简化成阶级斗争。
2. 不能替读者做具体人生决策。
3. 不能把"利益分析"变成"所有人都是自私的"。
4. 不能忽略人的情感和价值观因素。
5. 不能把历史结论直接套给现代人。

## 10. 不可写成什么

1. 不可写成"职场厚黑学"。
2. 不可写成"教你算计别人"。
3. 不可写成"所有人都是敌人"。
4. 不可写成"毛选万能论"。
5. 不可写成"只要分析利益就够了"。
