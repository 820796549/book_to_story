# 模型提炼：中国社会各阶级的分析

## 1. 模型名称：敌友识别模型

- 来源原文：中国社会各阶级的分析
- 对应栏目：底层解码篇、认知破局篇
- 模型要解决的问题：在复杂局势中，如何分辨谁是支持者、谁是反对者、谁是中间派。

### 原文依据

```text
"谁是我们的敌人？谁是我们的朋友？这个问题是革命的首要问题。"
"不可不将中国社会各阶级的经济地位及其对于革命的态度，作一个大概的分析。"
```

### 模型结构

```text
判断标准：经济地位（靠什么生存、有没有生产资料、生活水平如何）
-> 利益位置（变革会获益还是受损）
-> 政治态度（支持/观望/反对）
-> 策略安排（团结/争取/孤立/打击）
```

### 推理链

```text
现象：局势混乱，各方力量复杂
-> 问题：谁是支持者？谁是反对者？
-> 变量：经济地位（生产资料、收入来源、生活水平）
-> 判断：利益位置决定态度
-> 结论：支持者（利益相关）、反对者（利益受损）、中间派（利益不确定）
-> 行动含义：团结支持者、争取中间派、孤立反对者
```

### 使用场景

- 职场派系分析。
- 商业竞争中的利益相关者分析。
- 个人决策中的关系判断。
- 项目推进中的支持度评估。

### 失效边界

- 不能简化成"有钱就是坏人"。
- 不能忽略人的主观能动性和认知变化。
- 不能只分析不行动。
- 不能把现代复杂关系一刀切。

### 可转成文章的角度

- 底层解码篇：还原教员如何用这个模型判断敌友。
- 认知破局篇：教读者如何用这个模型分析职场/商业中的利益格局。

## 2. 模型名称：阶级分析模型

- 来源原文：中国社会各阶级的分析
- 对应栏目：底层解码篇
- 模型要解决的问题：如何系统性地分析一个群体中的不同力量。

### 原文依据

```text
原文对五大阶级的逐层分析：地主/买办 -> 中产阶级 -> 小资产阶级 -> 半无产阶级 -> 无产阶级。
```

### 模型结构

```text
第一步：分类（按经济地位划分群体）
第二步：分析每类的利益位置
第三步：推导每类的政治态度
第四步：制定针对每类的策略
```

### 推理链

```text
群体 -> 按经济地位分类 -> 每类的利益位置 -> 每类的态度 -> 综合策略
```

### 使用场景

- 组织变革中的利益相关者分析。
- 市场细分中的用户分类。
- 团队管理中的成员分析。

### 失效边界

- 分类不能太粗或太细。
- 不能忽略分类之间的动态变化。
- 不能只分类不行动。

## 3. 模型名称：力量判断模型

- 来源原文：中国社会各阶级的分析
- 对应栏目：底层解码篇
- 模型要解决的问题：在资源劣势下，如何判断真正的力量在哪里。

### 原文依据

```text
"工业无产阶级人数虽不多，却是中国新的生产力的代表者，是近代中国最进步的阶级。"
"他们所以能如此，第一个原因是集中。第二个原因是经济地位低下。"
```

### 模型结构

```text
不要只看表面力量（人数、口号、声势）
要看结构力量：
- 经济地位（利益关联程度）
- 组织程度（集中度、纪律性）
- 变革意愿（现状获益还是受损）
```

### 推理链

```text
表面力量（人数多、声势大）-> 不一定真实
-> 结构力量（经济地位 + 组织程度 + 变革意愿）-> 真实力量
-> 结论：真正的力量在利益受损最大、组织程度最高的人群中
```

### 使用场景

- 弱势方如何找到突破口。
- 资源少时如何判断该投入哪里。
- 竞争中如何识别真正的对手和盟友。

### 失效边界

- 不能只看结构不看时机。
- 不能忽略领导力和组织的作用。
- 不能把力量判断当成静态结论。
